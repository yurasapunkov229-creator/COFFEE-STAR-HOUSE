СТРУКТУРА:
  server.py           — Бэкенд Flask + SQLite (запускать первым)
  admin_app.py        — Программа администратора (запускает сервер)
  k_app.py            — Программа кухни
  o_app.py            — Программа официанта
  templates/menu.html — Красивый сайт меню (для клиентов)
  restaurant.db       — База данных (создаётся автоматически)

═══════════════════════════════════════════════════════════════
БЫСТРЫЙ СТАРТ:
═══════════════════════════════════════════════════════════════

1. Установите Python с python.org (галочка "Add to PATH"!)
2. Запустите УСТАНОВИТЬ.bat
3. Запустите ЗАПУСТИТЬ_СЕРВЕР.bat

Браузеры:
  Администратор: http://localhost:5000/admin
  Кухня:         http://localhost:5000/k
  Официант:      http://localhost:5000/o
  
Клиенты (телефоны в той же Wi-Fi сети):
  http://ВАШ_IP:5000   (узнайте IP через ipconfig)

═══════════════════════════════════════════════════════════════
КАК СОБРАТЬ .EXE ПРИЛОЖЕНИЯ:
═══════════════════════════════════════════════════════════════

1. Запустите УСТАНОВИТЬ.bat
2. Запустите СОБРАТЬ_EXE.bat (5-10 минут)
3. В папке dist\ появятся admin.exe, k.exe, o.exe
4. Скопируйте папки templates\ и static\ рядом с admin.exe
5. Запускайте admin.exe — он автоматически стартует сервер!
6. k.exe и o.exe подключаются к запущенному серверу

═══════════════════════════════════════════════════════════════
ПОТОК РАБОТЫ:
═══════════════════════════════════════════════════════════════

Клиент (телефон):
  1. Открывает сайт меню → выбирает блюда
  2. Вводит номер столика → оплачивает через Kaspi
  3. Нажимает "Я оплатил" → заказ отправляется на сервер

Администратор (admin.exe):
  4. Видит заказ с пометкой "💳 Ожидает оплаты"
  5. Нажимает "✅ Оплата" → "👨‍🍳 Кухня"

Кухня (k.exe):
  6. Видит тикет заказа → отмечает блюда
  7. Нажимает "✅ Готово"

Администратор:
  8. Видит "готово" → нажимает "🚶 Официанту"

Официант (o.exe):
  9. Забирает, несёт, нажимает "✅ Доставлено"

Клиент:
  10. Видит статус заказа в реальном времени!
